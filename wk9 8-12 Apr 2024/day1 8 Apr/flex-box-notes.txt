::::::FlexBOX:::::::

flexbox places content based on available space without manually entering width and position properties etc.

step 1: container creation (flex) & place elements inside it to to apply flex property.


**** step 2: flex-direction((by default row ), column) ---> VERY IMPORTANT for step 3

main axis - flex direction(if row then row is main axis)

cross axis - !flex direction(opposite if flex direction is row then column is cross axis and vice-versa.)


step 3: alignment(justify-content, align-items)

application of alignment on elements:

justify-content - applies on main axis
align-items - applies on cross axis

properties : 
	flex-start(default)
	flex-end 
	center
	space-between
	space-evenly 
	space-around
